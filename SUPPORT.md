# Support

Paige is in public alpha.

- Use the [README](./README.md) for local setup.
- Open a focused issue when documented behavior fails reproducibly.
- Use [private vulnerability reporting](https://github.com/peelar/paige/security/advisories/new)
  for security concerns.

There is no paid support or guaranteed response time. Please remove secrets,
tokens, private Slack messages, and sensitive provider context from public
issues.
